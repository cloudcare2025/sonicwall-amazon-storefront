Features

Enterprise capabilities for complete on-premises email protection

Capture ATP Integration

Suspicious attachments are sent to the Capture ATP cloud sandbox for multi-engine analysis, providing zero-day threat detection with RTDMI technology.

Policy-Based Encryption

Automatically encrypt outbound messages based on content, sender, recipient, or keyword policies. TLS enforcement ensures secure transport.

Data Loss Prevention

Outbound scanning identifies PII, financial data, healthcare records, and custom-defined sensitive content before it leaves the organization.

Anti-Spoofing

SPF, DKIM, and DMARC validation combined with header analysis and sender reputation scoring blocks impersonation and spoofing attempts.

LDAP/AD Integration

Native integration with Active Directory and LDAP for automatic user provisioning, group-based policies, and directory-harvesting attack prevention.

Centralized Management

Single-pane management console for multi-appliance environments. Unified policy, reporting, and quarantine management across all nodes.
